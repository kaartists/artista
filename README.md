# React + Node + MySQL Starter

## Backend

1. Go to `backend/`
2. Copy `.env.example` to `.env` and fill in your MySQL credentials.
3. Run:

   ```bash
   npm install
   npm run dev
   ```

Backend runs on `http://localhost:5000`.

## Frontend

1. Go to `frontend/`
2. Run:

   ```bash
   npm install
   npm run dev
   ```

Frontend runs on the Vite dev server (e.g., `http://localhost:5173`).
